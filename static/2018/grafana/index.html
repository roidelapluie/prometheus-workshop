    Grafana

    Download and run grafana

    1. Go to the Grafana website and download the latest stable release
    2. For this exercise we will use the Standalone Linux Binaries(64 Bit) 
      $ wget https://s3-us-west-2.amazonaws.com/grafana-releases/release/grafana-5.3.2.linux-amd64.tar.gz
$ tar -zxvf grafana-5.3.2.linux-amd64.tar.gz
$ cd grafana-5.3.2
$ ./bin/grafana-server
    3. Open Grafana in your browser
